# Admin Panel

An Admin dashboard for booking, account information and more with login and register functionalities.

## Features

- Login
- Register
- Reset password
- Booking
- Party add
- Booking view
- Party view
- Cross platform

## Tech Stack

**Client:** React, TailwindCSS, Daisyui

**Server:** Node, Express , MySql .
